Detailed Description

Represents a texture-based animation made of one or more frames.

Manages the animation state (current frame) and provides access to a sprite for rendering.

An animation can be a single frame or multiple frames arranged in the texture. The update() function changes the animation frame based on the defined speed.

Note
Animation speed cannot not be zero.
