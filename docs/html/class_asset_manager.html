Detailed Description

Centralized storage and retrieval of game assets.

The AssetManager is resposible for loading, storing and providing access to common game resources: textures, animations, fonts, sounds and musics.

Assets are identified by string key, allowing them to be retrieved anywhere in the code without reloading them. Trying to retrieve a non-stored asset will throw an error.

Each asset is loaded only once and remains available as long as the AssetManager exists.

Note
This class is non-copyable and non-movable to avoid duplication of resources.
