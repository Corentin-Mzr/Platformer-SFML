Detailed Description

Main application class, runs the game.

GameEngine manages the game loop, scenes, assets, config and window. It serves as the entry point and controller of the application.

Usage:

  • run(): Start the main game loop
  • update(): Update the game logic
  • quit(): Stop the game
  • change_scene(name, scene, end_current): Switch between scenes

The class provides access to:

  • AssetManager for textures, fonts, sounds, musics and animations
  • Window and view for rendering
  • Config data for window, levels, etc.
Note
Non copyable, non movable
Should be constructed from a TOML config file
Holds GameSettings for volume control
Manages scene lifecycle and user inputs
